# GhostMail

A feature-rich email client built with React, TypeScript, and Tailwind CSS. Provides a modern Gmail-style interface with compose, labels, search, snooze, and keyboard shortcuts.

## Features

- Gmail-style email management (inbox, sent, drafts, spam, trash)
- Compose modal with recipient, subject, and body
- Label management for organising emails
- Full-text search with filters
- Snooze emails for later
- Bulk select and actions (archive, delete, mark read, mark spam)
- Keyboard shortcuts for power users
- Command palette for quick navigation
- Google OAuth integration
- Dark/light theme (auto-detection)
- Inline reply without leaving the email view
- Onboarding flow for new users

## Tech Stack

- React 19, TypeScript, Vite
- Tailwind CSS 4 + `tailwind-merge` + `clsx`
- Zustand (state management)
- Framer Motion (animations)
- Lucide React (icons)
- Google OAuth / Gmail API

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
